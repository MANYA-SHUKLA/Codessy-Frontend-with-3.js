# Codessy Website

This is the website for Codessy made by MANYA SHUKLA, a creative agency specializing in web development, design, and branding. This project is built with Next.js, 3.js, typescript and features a modern, interactive, and animated user interface.

## âœ¨ Features

-   **Interactive 3D Globe:** A stunning 3D globe created with Three.js and React Three Fiber.
-   **Smooth Scrolling & Animations:** Seamless page transitions and animations powered by Framer Motion.
-   **Responsive Design:** Fully responsive layout for all devices, from mobile to desktop.
-   **Contact Form:** A functional contact form integrated with EmailJS.
-   **Modern UI/UX:** A clean and modern design with a focus on user experience.

## ğŸ“¦ Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

-   Node.js (v20 or later)
-   npm, yarn, or pnpm

### Installation

1.  Clone the repo
    ```sh
    git clone https://github.com/MANYA-SHUKLA/Codessy-Frontend-with-3.js.git
    ```
2.  Install NPM packages
    ```sh
    npm install
    ```
3.  Run the development server
    ```sh
    npm run dev
    ```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
